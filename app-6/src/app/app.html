***** ***** Pipes ***** *****

In Angular, pipes are a way to transform data in your templates without changing the actual data in your component.

Think of them as small helpers that format values for display.

Basic syntax: { value | pipeName }

You can also chain pipes: value | pipe1 | pipe2

Common Built-in Pipes

import 'commonModule' to use pipes

Angular provides several ready-to-use pipes:


today | date:'fullDate'

name | uppercase
name | lowercase

price | currency:'INR'

number | number:'1.2-2'

value | percent

obj | json

date-format: "dd-mm-yyyy" = {{date | date:"dd-mm-yyyy"}}

----------------------TimeZone----------------
date: "full":"utc" = {{date | date:"full":"utc"}}
date: "short':'Asia/Kolkata" = {{date | date:"short":"Asia/Kolkata"}}
-------------------Locale support---------------------
date: "fullDate":" ":"en-US" = {{date | date:"fullDate":" ":"en-US"}}
date: "fullDate":" ":"en-IN" = {{date | date:"fullDate":" ":"en-IN"}}

Custom Pipes

>>> ng generate pipe custom-pipe/trimText ([folder/file]name)

1. after creating, import the pipe to the target component(.ts)

to use the pipe, name of the pipe will be written in, the pipe(trimText) files @Pipe(name: trimText)

2. write the custom logic of the pipe inside transform method & return it
