from parent component: {{user}} {{meme}}

1. define @Input in child component(.ts)

@Input() user: string | undefined;

it may possible that 'user' is not defined in parent, hence string | undefined


2. Pass data from parent component(.html)

<app-child [user]= "userName"></app-child>

here, userName is the variable defined in parent(.ts)


3. Define the data in the parent component

userName="pranav Kumar"
