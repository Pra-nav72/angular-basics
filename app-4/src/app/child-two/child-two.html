In Angular, sending data from a child component to a parent component is typically done using event binding with @Output and EventEmitter.

  • The child component emits an event
  • parent component listens to that event


    • inside Child Component(.ts)

    @Output() dataEvent = new EventEmitter<string>();

    sendData() {
    this.dataEvent.emit('Hello from Child!');
    }

    inside Child Component(.html)

    to trigger the sendData(), need button
    <button (click)="sendData()($event)">send to parent</button>


    inside Parent Component(.html)

    <app-child (dataEvent)="receiveData($event)"></app-child>
    <p> { message } </p>

    inside Parent Component (.ts)

    message=' ';
    receiveData(data: string){
    this.message = data;
    }
