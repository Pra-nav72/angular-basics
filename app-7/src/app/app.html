Routing

1. create components (inside a folder: pages/home, about, contact)

2. register these inside routes file of app component (app.routes.ts)

[
{path:" ", component:Home},
{path:"contact", component:Contact},
{path:"about", component:About},
]

3.inside app componets (.ts) file,
import RouterOutlet - to view all routes components (Home, Contact...) added in point 2.
import RouterLink - to create the url link

4. Add the router-outlet inside app.html

5. to create navigation button, you need to use 'routerLink' instead of href=" " inside the <a> element.

If you use href="" instead of routerLink , then your page will reload when you switch between the pages.

with href=""

  • Home
  • Contact
  • About

with routerLink

  • Home
  • Contact
  • About

Page Not Found Error:

1. create & design the component for page not found

2. register it after all the component(lowest line) in app.routes.ts file

it should be look like: {page:" ** ", component:pageNotFound}

'**' means on any url request go to this corresponding component only.

That's why we use it at the last of the route description

To redirect (redirectTo):

to redirect, after any specific route url(lets say, /pranav), in the app.routes.ts, {page:"/pranav", redirectTo: '<url(/home, /about...etc)>' }

now you can see output based on url
